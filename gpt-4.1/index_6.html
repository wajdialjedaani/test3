Accessibility Help

This website meets WCAG 2.2 AAA standards. All content is keyboard accessible, has high contrast, and works with assistive technologies. Use "Skip to main content" for faster navigation.

  • All forms have proper labels and error feedback.
  • Search and navigation are available in multiple ways.
  • All interactive elements are accessible by keyboard and screen reader.
  • Use the theme switcher to adjust colors for your needs.

Search Recipes by Ingredient

Enter an ingredient to filter recipes (e.g. chicken, rice). Press Enter to search. Leave blank to show all recipes.

How to Use This Site

  1. Search: Use the search bar to find recipes by ingredient.
  2. Share: Submit your favorite recipe using the form.
  3. Browse: View all recipes below. Click stars to rate.
  4. Keyboard: All functions are keyboard accessible.
  5. Theme: Switch color themes below for your comfort.
