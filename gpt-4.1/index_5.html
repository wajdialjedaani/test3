Accessibility Information

This portfolio is designed to meet WCAG 2.2 AAA accessibility standards. It includes enhanced color contrast, keyboard navigation, ARIA labels, alternative text for all images, and multiple ways to locate content. If you encounter any accessibility barriers, please contact me.

  • All images have descriptive alt text.
  • All interactive elements are keyboard accessible and have visible focus indicators.
  • Color choices meet or exceed 7:1 contrast ratio.
  • Motion is minimized and respects your system's reduced motion settings.
  • Help and instructions are available throughout the site.

Help & Definitions

WCAG
WCAG is a set of guidelines for making web content more accessible to people with disabilities.
ARIA
ARIA is a set of attributes to improve accessibility of web content, especially dynamic content and advanced user interface controls.
AAA Level
The highest level of accessibility conformance, including advanced requirements for color contrast, keyboard navigation, and content alternatives.

For more definitions or help, please contact me.
